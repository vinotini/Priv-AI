# PrivAI - AI-powered Privacy Leak Detection

## Overview

PrivAI is a full-stack web app that detects privacy leaks in uploaded documents (.pdf, .docx, .txt). It uses AI-enhanced keyword detection and dynamic risk scoring to help users secure their data.

## Features

- Upload PDF, DOCX, or TXT files
- Extract text and detect sensitive info with AI-enhanced NLP
- Dynamic risk score visualization with animated meter
- Clean, pastel-themed responsive UI

## Setup

1. Install dependencies: `npm install`
2. Run locally with Vercel: `vercel dev`
3. Upload files and see real-time leak detection and risk score

## Future

- Advanced NLP models like Transformers
